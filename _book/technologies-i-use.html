Hackathon Information

Topic 1 Technologies I Use

Figure 1.1: A Programmer Working on Some Code

If you are going to work with me, then I think that it’s only fair that you know the tools that I typically use during a hackathon. Not only will this (hopefully) give you a rough idea of what to expect, but it’ll also give you an idea of what to read up on.

The technologies that I typically use during each hackathon are1:

  1. R
  2. Git
  3. Shiny
  4. R Markdown

However, most importantly is that you are acquainted with technologies 1. and 3. 2. and 4. are still somewhat optional;2 however, definitely read up on them nonetheless!

  1. Mind you - I don’t necessarily use all technologies during each event.↩︎

  2. Not knowing 2. and 4. would be like not knowing how to use a mixing stand. You can definitely still bake a cake without ever having to know how to use one, but knowing how to use a mixing stand to make your cake batter would definitely be a lot more convenient!↩︎
